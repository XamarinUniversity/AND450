Before you begin

This is an Android class. To complete the coding exercises, you will need an Android development environment with either an emulator or a device to run the exercise on. Please see the setup page if you need help installing the Xamarin development environment.

Overview

A Bindings Library is an assembly containing Managed Callable Wrappers for Java types. This course shows how to use the Xamarin tools to create Bindings Libraries for both .JAR and .AAR files. Note that this class covers the core concepts; it does not describe how to customize the generated bindings.

Objectives

  1. Determine which Java libraries can be bound
  2. Bind a .JAR
  3. Bind an .AAR
  4. Handle a private-use reference .JAR
  5. Handle a public-use reference .JAR
  6. Examine how Java-language constructs are mapped to C#

Lecture

AND450 - Building a Java Bindings Library.

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Bind a .JAR
    Exercise
    In this exercise, you will create a Bindings Library for a .JAR file and use the Managed Callable Wrappers in an Android app.
  2. Bind an .AAR
    Exercise
    Here, you will create a Bindings Library for an .AAR file and use the types and resources from the .AAR file in an Android app.
  3. Examine mappings for various Java constructs
    Exercise
    This exercise shows what C# is generated for various Java-language constructs.
